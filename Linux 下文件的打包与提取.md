# Linux 下文件的打包与提取

Linux下文件的打包压缩，和安装包打包。

## 压缩

### tar

tar -cvf xxx.tar xxx/

### zip

打包文件夹
zip -r mypac.zip ./* 
zip -r xxx.zip xxx/

打包文件

zip xxx. zip xxx

### gzip

gzip -r xxx.gzip xxx/

## 解压缩

tar -xvf xxx.tar



unzip xxx.zip

ungzip xxx.gzip
